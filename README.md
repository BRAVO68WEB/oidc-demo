# oidc-demo

![oidc](./openid.png)

Demo implementation of OIDC server

- [server](./server) - OIDC server implementation
- [demo](./demo) - OIDC client demo with [openid-client](https://npmjs.com/package/openid-client) library

## Usage

### Install dependencies

```bash
bun install
```

### Run

```bash
bun dev
```
